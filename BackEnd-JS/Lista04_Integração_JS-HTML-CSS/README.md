# Exercicios da Lista 4: Integração JavaScript, html e CSS!

**Ex.1 -** Crie botões no arquivo HTML que quando clicados exibam no console:
<ul>
    <li>Todos os dados de todos os idosos;</li>
    <li>Apenas os idosos vivos;</li>
    <li>Apenas os idosos mortos;</li>
    <li>Apenas os idosos 'jovens';</li>
</ul>


**Ex.2 -** Crie uma div no html e associe este elemento à uma variável no arquivo JS. Defina o texto interno desta div - no arquivo JS - comoo nome do idoso que ocupa a 6° posição do vetor idosos!

**Ex.3 -** Crie um elemento em JS e atribua-o a uma nova constante 'coisa'. Defina o texto interno deste novo elemento para 'Elemento criado' e estabeleca uma relação entre este elemento novo (filho) e uma div (pai) no html, de forma a exibir a mensagem no navegador.

**Ex.4 -** Componha uma string que contenha o nome, idade e situação de vida do idoso na posição 7 do vetor. Para tal use relações pai-filho entre o arquivo html e o arquivo JS.

**Ex.5 -** Crie uma função que passe por cada elemento do vetor, escrevendo a string do exercício 4 para cada um dos idosos. Para tal use relações pai-filho entre o arquivo html e o arquivo JS.

**Ex.6 -** Crie uma função que escreva na tela apenas os dados filtrados dos idosos maiores que 60 anos. Para tal use relações pai-filho entre o arquivo html e o arquivo JS.

**Ex.7 -** Revise os códigos e inclua pequenas estilizações em CSS.
